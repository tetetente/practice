焼肉屋で注文する際に使用するタブレットを想像して試作。

・まず、メソッド名の統一の無さと大文字小文字の適当さは素直にごめんなさい。

・Form1はメニュー画面、Form2は注文画面、Form3は履歴画面、Form4は会計画面 となっている。

・画像は全てパソコン内にあるImageフォルダ内に保存している。

・サーバーを介してデータの送受信を行うメソッドは、全てDAO.csにまとめている。

・DAO.csの一番上に記載している「seat」は一応席の番号で考えており、ここを2などに変更する事で席ごとに注文をわけられる。
